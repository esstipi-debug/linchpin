Linchpin Inventory AI

AI-generated reorder policy recommendations from your own confirmed sales history — a free, read-only dry run, right inside Odoo.

What it does

One click, under Inventory > Products > Linchpin Inventory Analysis, sends your confirmed sale order history to Linchpin's cloud analysis service and shows back, per SKU:

  • A demand forecast fit to your actual sales pattern
  • A recommended reorder point and reorder quantity ((s,Q) / (R,S) policy)
  • A safety-stock target at your chosen service level
  • A budget-fit inventory investment estimate

Nothing is written back to Odoo. This module only displays a recommendation for a human to review and act on — it never touches your stock levels, reorder points, or purchase orders.

What data is sent, and where

This module requires an external service to run. For each confirmed sale order line in the history window you choose, it sends: the product's SKU (or internal ID if no SKU is set), order date, quantity sold, unit cost, your primary supplier's lead time in days, and your company name — over HTTPS to Linchpin's cloud service, using the API key you enter in Settings > Inventory > Linchpin. No stock or on-hand quantities are ever read or sent. The exact same disclosure is shown again inside the wizard itself before you run an analysis, not just here.

Requirements

Because this module includes Python code that calls an external service, it runs on Odoo.sh (private repository) or an on-premise installation. It is not compatible with Odoo Online. Depends on the standard Inventory (stock) and Sales (sale) apps.

You'll also need a Linchpin API key, issued per client. A dry run like this one is free to try — enter your key in Settings > Inventory > Linchpin to get started.

Access control

This module adds its own Linchpin User permission group, separate from the standard Inventory Administrator role. Nobody has it by default — grant it explicitly, per user, from Settings > Users, since running an analysis means sharing product, cost, and sales data with an external service.

This is a dry run

This module is a free, read-only lead-in to Linchpin's full inventory optimization service — multi-echelon planning, DDMRP, safe-staged writeback straight into Odoo's own reorder points and draft purchase orders, and more — available directly, outside the Odoo Apps Store.
